Css Selectors

  1. Звернутися до CSS елемента можна за назвою елемента.
    Наприклад:
    p { /* Стилізація для всіх елементів p (параграфів) */ }
  2. Звернутися до CSS елемента можна за id.
    Наприклад:
    #header { /* Стилізація для елемента з ідентифікатором "header" */ }
  3. Звернутися до CSS елемента можна за класом.
    Наприклад:
    .button { /* Стилізація для всіх елементів з класом "button" */ }
  4. Звернутися до CSS елемента можна за декількома класами.
    Наприклад:
    .red-text.bold { /* Стилізація для елементів з класами "red-text" та "bold" */ }
  5. Звернутися до CSS елемента можна за вкладеністю.
    Наприклад:
    header nav { /* Стилізація для елементів nav, які є дочірніми для header */ }
